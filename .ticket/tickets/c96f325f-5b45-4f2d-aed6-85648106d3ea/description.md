# Goal
Refocus the audit agent guidance on executing audit tools against the requested target context, surfacing findings, and summarizing results in a canonical findings-and-recommendations format.

## Scope
- update the canonical rule source for audit instructions
- regenerate the managed audit instruction output
- validate the regenerated guidance reflects the new execution and summary emphasis

## Acceptance
- audit instructions tell agents to run the audit tool on the target context instead of only describing the tool surface
- instructions define a canonical summary shape that separates findings from recommendations
- managed outputs are regenerated from the rule source
